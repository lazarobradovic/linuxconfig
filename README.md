# FICHIERS DE CONFIGURATIONS LINUX

Personal Linux .config and .themes files.

## Composants

### **.CONFIG - Elements for X11**
- **bspwm** : Windows tiling manager
- **sxhkd** : Shortcuts
- **polybar** : Status bar
- **conky** : Widgets for the desktop
- **rofi** : Window switcher, run dialog, ssh-launcher and dmenu

### **.CONFIG - Elements for Wayland**
- **hypr** : Contains hyprland (window manager) and hyprpaper (wallpaper)
- **wofi** : Launcher/menu, though in the last update I switched to rofi again
- **waybar** : Used the config created by https://github.com/Alexays/Waybar

### **.CONFIG - IDE**
- **neovim** : Contains init.lua file to implement basic configuraiton and plugins 

### **.CONFIG - Terminals**
- **alacritty** : Used for both X11 and Wayland
- **kitty** : Used by some scripts of Waybar under Hyprland

### **.CONFIG - Others**
- **dunst** : Notification daemon
- **scripts** : Personal scripts

### **.THEMES**
- **oomox-space-wallpaper** : GTK theme matching the wallpaper
