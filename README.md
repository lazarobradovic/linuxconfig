# FICHIERS DE CONFIGURATIONS LINUX

Configuration personnelle pour bspwm, sxhkd, polybar, conky et autres composants utiles.

## Composants

- **.CONFIG**
-- **bspwm** : Windows tiling manager
-- **sxhkd** : Shortcuts
-- **polybar** : Status bar
-- **conky** : Widgets for the desktop
- **.THEMES**
-- **oomox-space-wallpaper** : GTK theme matching the wallpaper
